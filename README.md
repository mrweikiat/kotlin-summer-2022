<a name="drawing_app"></a>
## Drawing App

- Features:
  - Drawing on a blank canvas
  - Uploading an image and drawing over it
  - undo and redo feature for drawing errors
  - eraser 
  - color palette 
  - saving to local machine
  - sharing to friends on gmail, email, link etc 

- Concepts adopted:
  - Coroutines
  - UI design using xml
  - Playing around with Android permissions for read/write to storage

## WorkOut App
- Features:
  - 10 HIIT Exercises: 30s Per exercise followed by 10s rest
  - BMI Calculator to track your fitness and health 
  - Display history for past attempts

- Concepts adopted:
  - Text To Speech
  - Progress Bar
  - View binding
  - RecyclerViews
  - Room database
